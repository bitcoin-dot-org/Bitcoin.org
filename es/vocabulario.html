Algunas palabras de Bitcoin que puedes escuchar

Bitcoin ofrece un nuevo enfoque a los pagos y, como tal, hay algunas palabras nuevas que podrían convertirse en una parte de tu vocabulario. No te preocupes, ¡incluso la televisión crea nuevas palabras!

Dirección

Una dirección Bitcoin es como una dirección fisica o un email. Es la única información que debe proporcionar para que alguien yr pague con Bitcoin.

Cadena de Bloques

La Cadena de Bloques es un registro publico de todas las transacciones de Bitcoin, en orden cronologico. La Cadena de Bloques se comparte entre todos los usuarios de Bitcoin. Se utiliza para verificar el balance de direcciones Bitcoin y para prevenir el doble gasto.

Bloque

Un bloque es un registro en la cadena de bloques que contiene confirmaciones de transacciones pendientes. Aproximadamente cada 10 minutos en promedio, un nuevo bloque incluyendo las transacciones se anexa a la Cadena de Bloques a traves de la mineria.

BTC

BTC es el código de tres letras no oficial de Bitcoin. Puede ser utilizado como una abreviatura, como USD para el dólar estadounidense.

Confirmacion

La confirmacion significa que una transaccion ha sido verificada por la red y es muy poco probable que se invierta. Una confirmación es bastante seguro. A pesar de las grandes cantidades (por ejemplo, 1000 $ USD o más), se puede esperar que una transacción tenga más confirmaciones - 6 es el número con frecuencia elegido. Cada nueva confirmación, disminuye el riesgo de un retroceso exponencial.

Criptografia

La criptografía es la rama de las matemáticas que nos permite crear pruebas matemáticas que proporcionan altos niveles de seguridad. El comercio en línea y los bancos ya utilizan criptografía. En el caso de Bitcoin, la criptografía se utiliza para hacer imposible que alguien pueda gastar los fondos del monedero de otro usuario o para corromper la cadena de bloques. También puede ser utilizada para cifrar un monedero, de manera que no se pueda utilizar sin una contraseña.

Doble Gasto

Si un usuario malintencionado intenta pasar sus bitcoins a dos destinatarios diferentes al mismo tiempo, se produce un gasto doble. La minería de Bitcoin y la cadena de bloques están ahí para crear un consenso en la red acerca de cuál de las dos transacciones ganará.

Tasa de Hash

La tasa de hash es la unidad de medida del poder de procesamiento de la red Bitcoin. La red Bitcoin debe hacer uso intensivo de operaciones matemáticas por razones de seguridad. Cuando la red alcanza una tasa de hash de 10 TH / s, esto significa que puede hacer diez billones de cálculos por segundo.

Mineria

La mineria en Bitcoin es el proceso de realizar calculos matematicos mediante computadoras para confirmar transacciones en la red Bitcoin e incrementar la seguridad. Como recompensa por sus servicios, los mineros Bitcoin puede cobrar costos de transacción para las transacciones que confirman junto con bitcoins nuevos bitcoins que se crean. La minería es un mercado especializado y competitivo en el que los beneficios se reparten de acuerdo a la cantidad de cálculo que se hace. No todos los usuarios de Bitcoin realizan minería y no es una manera fácil de hacer dinero.

P2P

Punto a punto se refiere a los sistemas que trabajan como una organizacion colectiva permitiendo que cada individuo interactue directamente con otros. En el caso de Bitcoin, la red se construye de tal manera que cada usuario está transmitiendo transacciones de otros usuarios. Y algo muy importante, ningún banco se requiere como intermediario.

Clave Privada

Una clave privada es una parte secreta de los datos que acredite su derecho a utilizar una dirección Bitcoin para transferirlo a otra a través de una firma criptográfica. Cada direccion Bitcoin tiene su propia clave privada única. Tus claves privadas se almacenan en tu computadora si utilizas un software de monedero, mientras que se encuentran almacenadas servidores remotos si utilizas un monedero web. Las claves privadas nunca debe ser revelado, ya que permiten que se puedan transferir bitcoins desde dichas direcciones.

Signature

Una firma criptografica es un mecanismo matematico que permite a alguien probar demostrar su propiedad. En el caso de Bitcoin, una direccion Bitcoin y su clave privada estan vinculadas por un tipo de magia matematica. Cuando tu software de Bitcoin firma una transacción con la clave privada correspondiente, toda la red puede ver que la firma coincide con la dirección Bitcoin. Sin embargo, no hay forma de que el mundo descubra la clave privada para robar tus bitcoins.

Monedero

Un monedero Bitcoin se refiere a el equivalente a un monedero fisico en la red Bitcoin. Cada monedero Bitcoin puede mostrar el saldo total de todas las direcciones Bitcoin que contiene. Al igual que puedes contar con el dinero en tu monedero real. Y de la misma manera, un monedero Bitcoin te permite pagar una cantidad específica a una persona específica. Esto es diferente a las tarjetas de crédito en la que se paga por mercado.
