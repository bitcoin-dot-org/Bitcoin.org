Developer Documentation

As a peer-to-peer digital currency, Bitcoin is designed to be mostly independent of any central authority. This documentation is for developers who want to build programs that interact with the Bitcoin network such as wallets, merchant systems, and specialized applications.

P2Pool and Sharechain

P2Pool (Peer-to-Peer Pool) is a decentralized mining pool that operates without a central authority. Instead of relying on a pool operator, P2Pool creates its own blockchain-like data structure called a "sharechain" that is separate from the main Bitcoin blockchain.

How P2Pool Works

P2Pool operates by creating a peer-to-peer network of mining nodes that distribute work and share rewards among participants. Each participating node maintains a complete copy of the sharechain, which contains records of all mining attempts (shares) submitted by all participants in the pool.

Sharechain Details

The sharechain is a separate blockchain maintained by the P2Pool network that tracks mining attempts and rewards distribution. Key characteristics include:

  • Decentralized reward distribution: Miners are rewarded proportionally to their contributed hashpower for shares found in the sharechain
  • Dynamic difficulty adjustment: The sharechain automatically adjusts difficulty to maintain consistent block timing appropriate for the participating hashrate
  • Transparent merged mining: P2Pool supports merged mining with other cryptocurrencies, allowing miners to simultaneously mine Bitcoin and other compatible coins
  • Enhanced privacy: P2Pool provides better transaction privacy compared to traditional pools by avoiding central points of failure

Technical Implementation

P2Pool implementation involves:

  • Running a separate P2P network node that connects to other P2Pool nodes
  • Each node maintains a complete sharechain copy for reward verification
  • Shares are distributed based on measured hashpower contribution
  • Block propagation uses the sharechain for faster, more reliable transaction inclusion

Benefits

  • Reduced orphan rates: P2Pool's built-in relay network historically has had lower orphan rates than traditional pools
  • Address privacy: P2Pool provides enhanced privacy over other mining pools through decentralized payout distribution
  • Merged mining capability: P2Pool can transparently engage in merged mining without sharechain enhancement
